Handson - Array Of Objects

Question:

let studentRecords = [ {name: 'John', id: 123, marks : 98 },
{name: 'Baba', id: 101, marks : 23 },
{name: 'yaga', id: 200, marks : 45 },
{name: 'Wick', id: 115, marks : 75 } ]

Q1. We are interested in retrieving only the students' names; all the names should be in caps.
['JOHN', 'BABA', 'YAGA', 'WICK']

Answer:

        let ans = studentRecords.map((item) => item.name.toUpperCase());
        console.log(ans);

Output:

[ 'JOHN', 'BABA', 'YAGA', 'WICK' ]

Q2. Suppose we have the same dataset as above but this time we want to get the details of students who scored more than 50 marks.
[{name: 'John', id: 123, marks: 98 },{name: 'Wick', id: 115, marks: 75 }]

Answer:

        let ans2 = studentRecords.filter((item) => item.marks > 50);
        console.log(ans2);

Output:

        [
            { name: 'John', id: 123, marks: 98 },
            { name: 'Wick', id: 115, marks: 75 } 
        ]

Q3. Retrieve the details of students who scored more than 50 marks and have IDs greater than 120.

Answer:

        let ans3 = studentRecords.filter((item) => item.marks > 50 && item.id > 120);
        console.log(ans3);

Output:

        [ { name: 'John', id: 123, marks: 98 } ]

Q4. Consider the same scenario we have discussed above, but this time we would like to know the sum total of the marks of the students.

Answer:

        let sum = 0;
        let ans4 = studentRecords.filter((item) => item.marks)
            .map((item) => sum += item.marks);
        console.log(ans4[ans4.length - 1]);

Output:

241

Q5. This time we want to get only the names of the students who scored more than 50 marks from the same dataset used above.

Answer:

        let ans2 = studentRecords.filter((item) => item.marks > 50);
        console.log(ans2.map((item) => item.name));

Output:

[ 'John', 'Wick' ]

Q6. This time we are required to print the sum of marks of the students with id > 120.

Answer:

        let sums = 0;
        let ans6 = studentRecords.filter((item) => item.id > 120)
            .map((item) => sums += item.marks);
        console.log(ans6[ans6.length - 1]);

Output:

143

Q7. This time we are required to print the total marks of the students with marks greater than 50 after a grace of 15 marks has been added to those students who scored less than 50.

Answer:

        let mark = 0;
        let studentupdated = studentRecords.map((item) => {
            if (item.marks < 50)
                item.marks += 15;
            return item;
        })
        console.log(studentupdated);
        console.log(studentupdated.map((item) => item.marks > 50 ? mark += item.marks : null))
        console.log(mark);

Output:

        [
            { name: 'John', id: 123, marks: 98 },
            { name: 'Baba', id: 101, marks: 38 },
            { name: 'yaga', id: 200, marks: 60 },
            { name: 'Wick', id: 115, marks: 75 }
        ]       
        [ 98, null, 158, 233 ]
        233
